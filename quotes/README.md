# Fire Stick Hub

## Setup Instructions

1. **Photos**: Add small images to photos/ and list filenames in config.js.
2. **Quotes**: Add JSON file in quotes/quotes.json.
3. **Calendar**: Paste iCal URL in config.js.
4. **Meal Plan**: Add CSV URL in config.js.
5. **To Do**: Add iCal URL in config.js.
6. **Weather**: Insert API key and location in config.js.
7. **Customization**: Adjust fonts, colors, rotation times in config.js.
8. **Upload**: Upload folder to GitHub, enable Pages, open URL in Fire Stick browser. do i paste these to text docs
